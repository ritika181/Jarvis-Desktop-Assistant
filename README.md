# Jarvis Desktop Assistant

A Python-based AI Desktop Assistant inspired by Jarvis from Iron Man.  
This project can listen, execute tasks, and interact with the user using voice commands.

---

## Features

- Voice recognition and speech output
- Open websites, applications, and files
- Read PDFs and provide information
- Execute system commands
- Customizable task execution

---
